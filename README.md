# MOIS_MUR_IDE
Курсовая работа по дисциплине МОИС 

Инструкция по использованию системы управления АНПА
1. Требования и установка
На серверной стороне (подводный аппарат)
Установленный Python 3.7+

Библиотеки:

bash
pip install pymurapi opencv-python
Файл server.py (исправленная версия)

На клиентской стороне (компьютер оператора)
Установленный Python 3.7+

Библиотеки:

bash
pip install pillow pillow
Файл client.py (исправленная версия с клавиатурой)

2. Запуск сервера
На борту АНПА выполните:

bash
python server.py
Вы увидите:

text
MUR initialized
Server listening on 0.0.0.0:5000
Waiting for client connection...
Сервер готов к подключению одного клиента.

Важно: Сервер работает в цикле – после отключения клиента он снова переходит в режим ожидания. Перезапускать сервер не требуется.

3. Запуск клиента
На вашем компьютере:

bash
python client.py
Откроется графическое окно размером 1200×800 пикселей.

4. Подключение к серверу
Введите IP-адрес сервера (например, 192.168.1.100 или 127.0.0.1 для локального теста).

Введите порт – по умолчанию 5000.

Нажмите кнопку Connect.

При успешном подключении:

Интерфейс управления разблокируется.

Начнут отображаться видеопотоки с передней и нижней камер.

Появятся данные телеметрии (глубина, рыскание, тангаж, крен).

5. Управление аппаратом
5.1 Ручной режим (Manual)
По умолчанию вы находитесь в ручном режиме.

Управление возможно двумя способами:

Через мышь – кнопки на панели Manual Control:
Кнопка	Действие
Forward	Движение вперёд (оба движителя)
Backward	Движение назад
Up	Всплытие (вертикальные моторы)
Down	Погружение
Left	Поворот влево (горизонтальный мотор)
Right	Поворот вправо
Rotate left	Разворот на месте против часовой
Rotate right	Разворот на месте по часовой
При нажатии кнопки – аппарат начинает движение.

При отпускании кнопки – аппарат останавливается.

Через клавиатуру (активна только в ручном режиме, когда окно клиента в фокусе):
Клавиша	Действие
W	Движение вперёд
A	Поворот влево (горизонтальный мотор)
S	Погружение (вниз)
D	Поворот вправо
Q	Разворот на месте против часовой
E	Разворот на месте по часовой
Можно зажимать несколько клавиш одновременно – движения суммируются (например, W + D = вперёд + правый поворот).

При отпускании всех клавиш – полная остановка.

Клавиши не работают, если фокус находится в полях ввода IP/порта (чтобы можно было спокойно набирать адрес).

5.2 Автоматический режим (Auto)
Переключите радиокнопку на панели Mode в положение Auto.

Аппарат выполнит заранее запрограммированный сценарий:

Движение вперёд 3 секунды.
Остановка на 1 секунду.
Поворот вправо 2 секунды.
Остановка на 1 секунду.
Возврат к шагу 1.
В автоматическом режиме кнопки управления мышью и клавиши блокируются (движение определяет только программа).

Чтобы прервать авторежим и вернуть ручное управление, нажмите EMERGENCY STOP или переключите режим обратно в Manual.

6. Экстренная остановка
Кнопка EMERGENCY STOP (красная, большая) – немедленно останавливает все моторы и переключает режим в Manual.

Работает в любом состоянии (даже в авторежиме).

7. Отключение и переподключение
Отключение от сервера
Нажмите кнопку Disconnect.

Двигатели будут остановлены, видеопоток и телеметрия пропадут.

Сервер перейдёт в режим ожидания нового клиента.

Повторное подключение
Не нужно перезапускать ни клиент, ни сервер.

Просто снова нажмите Connect (при необходимости изменив IP/порт).

Если сервер работает, соединение восстановится.

Если при отключении клиента вы видите в консоли сервера сообщения Connection lost – это нормально. Сервер готов к новому подключению.

8. Телеметрия и видео
Телеметрия обновляется 10 раз в секунду: глубина (м), углы рыскания, тангажа, крена (градусы).

Видеопоток с двух камер: передней и нижней. Кадры сжимаются JPEG (качество 70), масштабируются до ширины 640 пикселей.

Частота обновления видео – около 10 кадров/с, GUI клиента обновляется до 20 кадров/с.

9. Завершение работы
На клиенте: закройте окно или нажмите Disconnect.

На сервере: нажмите Ctrl+C в терминале.

10. Примечания для разработчиков
Формат команд – JSON, разделитель \n.

Сервер принимает моторные мощности как массив из 5 целых чисел от -100 до 100.

Можно изменять автоматический сценарий в функции auto_control_loop() сервера.

Клавиатурные привязки легко переназначаются в словаре self.key_motor_map клиента.

Уачных погружений!
